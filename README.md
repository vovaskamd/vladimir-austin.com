# Vladimir Austin - Photography Website

This is a static version of the website, cleaned up from WordPress bloat and ready for static hosting.

## Repository Information
- **GitHub Repo:** [https://github.com/vovaskamd/vladimir-austin](https://github.com/vovaskamd/vladimir-austin)
- **Status:** Private Repository

## Local Setup
The local git configuration is set up to authenticate using a Personal Access Token (PAT).
You can push changes using standard git commands:

```bash
git add .
git commit -m "Description of changes"
git push
```

## Hosting
This site is ready to be deployed to static hosting providers like:
- **Netlify**
- **Vercel**
- **Cloudflare Pages**

Simply connect the GitHub repository to one of these services for automatic deployments.
